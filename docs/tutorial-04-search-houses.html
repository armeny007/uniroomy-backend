Search Description

Search Query

To search houses use GraphQL query:

{
  search_houses ( 
    university_id!
    min_monthly_budget
    max_monthly_budget
    min_bedrooms
    max_bedrooms
    commute_method
    max_commute_time
  )
  {
    house { 
      _id
      address
      ...
    }
  }
}

The query returns house list filtered by given criteria and sorted by House Rank
